    Summary

    快速鍵：在 function 上方一行輸入「///」自動產生

    • summary
      • <a href="xxx">123</a> => 檢視 function 時出現連結

        • (圖片內網址：ID驗證系列｜公司統一編號驗 | 辛西亞的技能樹)
      • <br/> => 檢視時訊息換行
      • 註解內含「&」的話，檢視註解時會顯示不出來
